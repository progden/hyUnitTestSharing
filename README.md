# hyUnitTestSharing
###Hung-Yang Tech Unit Test Sharing

環境

* 使用 Eclipse Luna
* Maven 3.3
* JDK 1.7
* MySQL
* Git

說明

* /readme.md 說明文件
* /create.sql 建立資料庫
* /unittest 範例程式，目錄使用 maven 預設結構

步驟

* 安裝資料庫
	* 建立 IssueTracker 資料庫，並建立帳號密碼預設使用 issuetracker/issuetracker
	* 匯入 create.sql
* 開啟 Eclipse
	* import Maven 專案
	* Maven -> update project
	* Run as -> Run on Server...
